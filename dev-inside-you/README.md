# dev-inside-you

This is gradle based learning project along with node js and typescript
config to setup aws services


1. install typescript=>
npm install typescript --save-dev

The compiler is installed in the node_modules directory and can be run with: npx tsc

2. Configuring the compiler=>
   npx tsc --init

3. installing aws cdk

   npm install -g aws-cdk

4. glue installation
   npm i @aws-cdk/aws-glue-aplha
